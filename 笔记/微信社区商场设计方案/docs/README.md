# 📚 微信社区商场小程序技术文档中心

## 🎯 文档概述

本文档中心为微信社区商场小程序项目提供完整的技术文档体系，基于**微信小程序原生开发 + 云开发**的技术架构，为不同角色的开发者提供专业、实用的技术指南。

### 技术架构
- **前端**: 微信小程序原生开发 (WXML/WXSS/JavaScript)
- **后端**: 微信小程序云开发 (云函数 + 云数据库 + 云存储)
- **数据库**: MongoDB 云数据库
- **部署**: 云开发一键部署

## 📋 文档分类说明

### 🔥 主要方案文档 (推荐使用)
基于微信小程序云开发的技术方案，为项目的**主要实施方案**。

### 📖 参考方案文档 (仅供参考)
传统开发方案的技术文档，仅作为**技术对比和学习参考**。

## 📖 文档系列详细说明

### 🔌 API接口文档系列

#### 📄 [用户相关API](api/user.md) 🔥
- **主要用途**: 用户认证、个人信息管理、权限控制
- **目标受众**: 前端开发者、后端开发者
- **核心内容**: 16个用户相关接口，包括微信登录、用户信息CRUD、权限验证
- **使用时机**: 开发用户认证和个人中心功能时
- **关联文档**: [云数据库设计](database/cloud-schema.md)、[编码规范](development/coding-standards.md)

#### 📄 [教务相关API](api/academic.md) 🔥
- **主要用途**: 课程表、成绩查询、考试安排、学籍管理
- **目标受众**: 前端开发者、后端开发者
- **核心内容**: 13个教务相关接口，涵盖学生学习生活的核心功能
- **使用时机**: 开发教务系统模块时
- **关联文档**: [云数据库设计](database/cloud-schema.md)

#### 📄 [商城相关API](api/mall.md) 🔥
- **主要用途**: 商品管理、订单处理、支付集成、商家管理
- **目标受众**: 前端开发者、后端开发者、产品经理
- **核心内容**: 30个商城相关接口，完整的电商功能体系
- **使用时机**: 开发商城系统模块时
- **关联文档**: [云数据库设计](database/cloud-schema.md)、[部署流程](deployment/deploy.md)

#### 📄 [论坛相关API](api/forum.md) 🔥
- **主要用途**: 帖子发布、评论互动、版块管理、搜索功能
- **目标受众**: 前端开发者、后端开发者
- **核心内容**: 28个论坛相关接口，完整的社区交流功能
- **使用时机**: 开发论坛系统模块时
- **关联文档**: [云数据库设计](database/cloud-schema.md)

## 🚀 快速开始指南

### 📋 新手开发者入门路径

#### 第一步：环境准备 (预计1-2小时)
1. 阅读 [项目README](../README.md) 了解项目概况
2. 按照 [云开发环境搭建](deployment/cloud-setup.md) 配置开发环境
3. 熟悉 [编码规范](development/coding-standards.md) 的基本要求

#### 第二步：技术理解 (预计2-3小时)
1. 学习 [云数据库结构设计](database/cloud-schema.md) 了解数据模型
2. 浏览 [用户相关API](api/user.md) 理解基础功能
3. 查看 [测试指南](development/testing.md) 了解质量标准

#### 第三步：实践开发 (预计4-6小时)
1. 实现第一个云函数（建议从用户登录开始）
2. 创建第一个小程序页面
3. 编写第一个单元测试
